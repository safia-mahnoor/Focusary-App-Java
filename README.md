 Focusary - Java OOP Productivity Timer App

Focusary is a Java-based productivity timer application inspired by the Pomodoro technique. It helps users stay focused by switching between focus sessions, short breaks, and long breaks through a simple graphical user interface.

 Features

- Focus timer with 25-minute session
- Short break and long break modes
- Start, pause, reset, and stop timer controls
- Background selection option
- Task management section
- Music play and stop controls
- Motivational quote display
- GUI-based interface

 Technologies Used

- Java
- Object-Oriented Programming
- Java Swing / GUI
- Event handling
- Timer-based functionality

 OOP Concepts Used

- Classes and objects
- Encapsulation
- Inheritance
- Polymorphism
- Event-driven programming
- Modular code structure
